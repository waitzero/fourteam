# 4조 입니다
플레이데이터(playdata) 13기 first mini project 

아재개그 판별 게임입니당

##  프로젝트 주제 소개
 Java에서 MVC Pattern, OOP, MySql 등을 활용하여 아재개그 판별 코드를 작성해 보았습니다.
 
 게임의 Stage는 총 5개로 구성되어있으며, 각 Stage당 20개의 문제가 랜덤으로 출제됩니다.
